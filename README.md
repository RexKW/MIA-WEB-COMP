# LOCA
Loca adalah platform all-in-one untuk mendukung UMKM di Indonesia. Dengan Loca, pengguna bisa menemukan UMKM beserta lokasi lengkapnya di peta, melihat katalog produk, dan mengakses dokumentasi media berupa video review dari konten sosial media, yang membantu membangun kepercayaan dan exposure bagi toko tersebut. Loca hadir untuk mempermudah pelanggan dan sekaligus mengangkat branding UMKM secara maksimal.

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- React Leaflet
- shadcn/ui
- gsap

## Features
- Peta interaktif untuk menemukan UMKM
- Katalog produk lengkap dengan deskripsi dan harga
- Dokumentasi media berupa video review dari konten sosial media
